This repository contains the projects I completed during my **Python Development Internship** at **SaiKet Systems**. The internship provided hands-on experience in Python programming by focusing on real-world applications, problem-solving, and software development fundamentals.

# 📂 Projects Included

## ✅ Task 1 – To-Do List Application

### Description
A command-line application that allows users to manage daily tasks by adding, viewing, updating, marking tasks as completed, and deleting tasks.

### Skills Learned
- Python Basics
- Lists & Dictionaries
- Conditional Statements
- Loops
- Functions

---

## ✅ Task 2 – Guess the Number Game

### Description
An interactive game where the computer generates a random number, and the player must guess it with higher/lower hints until the correct number is found.

### Skills Learned
- Random Module
- Loops
- Conditional Statements
- User Input Handling

---

## ✅ Task 3 – Basic File Handling

### Description
A Python application that reads text files, performs find-and-replace operations, saves the updated content, and handles file-related exceptions.

### Skills Learned
- File Handling
- String Manipulation
- Exception Handling
- Text Processing

---

## ✅ Task 4 – Basic Web Scraper

### Description
A web scraping application developed using **Requests** and **BeautifulSoup** to extract useful information from web pages.

### Skills Learned
- Web Scraping
- BeautifulSoup
- Requests Library
- HTML Parsing
- Data Extraction

---

## ✅ Task 5 – Currency Converter

### Description
A currency conversion application that retrieves real-time exchange rates through an API and converts currencies based on user input.

### Skills Learned
- API Integration
- JSON Handling
- User Input
- Error Handling

---

## ✅ Task 6 – Word Count Tool

### Description
A text analysis application that counts words, lines, characters, and displays the frequency of commonly used words in a text file.

### Skills Learned
- File Handling
- Dictionaries
- Text Analysis
- String Processing
